Quickgit

Quickgit is designed to fix problems and make Git easier for daily command line use. All of this is open source and available on GitHub.

Requirements

Required

  • A Linux/UNIX based operating system.
  • Git-Core

Recommended

  • SSH Repo remotes (as opposed to HTTP remotes)
  • GitHub repos (most commands work best using GitHub)

Updates

I'm updating Quickgit all the time when I need to do a lot with Git often, so I'll push an update to a new branch every time I consider an update stable for release.

Installation

Quickgit can be ran via exporting the PATH variable to the downloaded location for now, a better solution is being worked on.

Note: "quickgit changelog" is only functional with all QuickGit binaries located at /quickgit. This will be changed to be better in the future.
